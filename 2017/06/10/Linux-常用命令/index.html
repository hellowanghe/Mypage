常用基本命令（Centos7）

# ls 查看当前目录下文件和文件夹

# ll 查看当前目录下文件和文件夹的详细信息

# ls -lh 文件和文件的大小一k,M显示，不加h是以字节显示（-lhr正序排列，-lhv倒序排列）

# cd /home 改变目录到绝对路径的home目录下（/表示绝对定位）

# cd home 到当前目录下的home路径

# cp 1.txt /home/jack 复制名叫1.txt的文件到/home/jack目录下

# cp one.txt /home/jack/two.txt 复制名叫one.txt的文件到/home/jack下并重命名为two.txt

# mv one.txt three.txt 将名为one.txt的文件重命名为three.txt

# mv one.txt /root 将名为one.txt的文件移动到/root目录下

# mkdir tom 新建名为tom的目录（文件夹）

# touch main.java 新建名为main的java格式文件

# touch book.txt 新建名为book的txt格式文件

# rm one.txt 删除名为one.txt的文件，（rm意为remove）

# rmdir tom 删除名为tom的文件夹，只能删除空文件夹（同 # rm -d）

# rm -r tom （recursive递归删除文件夹，及其包含的内容，每次删除都会有提示。

# rm -rf tom 删除文件夹，强制删除文件夹中所有内容，不再询问
